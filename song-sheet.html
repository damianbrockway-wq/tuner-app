The strum
Easy version: one down-strum on each beat (D D D D). When ready: the island strum below — down, down-up, up-down-up. Start slow; keeping time matters more than speed.
How it's built
Tips
It's the same four chords all the way through — C, G, Am, F — looping. The trickiest change is C → G (three fingers move); drill that one in Smart Practice. Let chords ring; don't rush the changes. Once the loop feels easy, sing along.

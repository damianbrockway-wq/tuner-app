# Pluck — Game / Juice Ideas (80s Synthwave Arcade)

Working memory for the "make it addictive, like a game" direction.
North star: 80s SYNTHWAVE ARCADE — neon magenta/cyan, perspective grids, VHS scanlines,
chrome text, pixel/particle bursts, cheesy synth stings. Habit-forming by celebrating real
progress, NOT dark patterns (learning app, incl. kids).

## Fireworks & juice — where the bursts go
- Badge earned -> badge flies in, gold fireworks, chime sting (make it a MOMENT)
- Lesson complete -> confetti + rank bar fills with a clunk
- New personal best (changes/min, timing %) -> "NEW BEST!" gold burst
- Streak day -> flame grows, "Day N" pop
- Song finished -> stage lights sweep + tiny crowd cheer
- Tuner lock-in -> needle spring-snap + shimmer

## Combos — the arcade hook (high priority)
- Chain clean chords with no miss -> combo counter (2x,3x,5x), chord name grows,
  colors intensify, success chime climbs a pitch ladder; miss = soft reset
- Full-combo song/drill -> "PERFECT RUN" mega-fireworks

## Hidden easter eggs (discovery delight)
- [DONE] Rick Roll — tap the tuner logo 7x -> 80s synthwave overlay
- Tap Pluck logo 5x -> rainbow confetti + secret message
- Tune dead-on A440, hold 3s -> tuning-fork animation + "perfect pitch"
- Secret chord cantrip: play C-G-Am-F slowly -> unlock a bonus theme
- After 10th song -> "Rockstar Mode": stage lights, bigger bursts, crowd noise
- Date eggs: gentle snow in December, etc.

## Come-back loop (healthy-addictive)
- Streak with forgiving freeze (don't punish one missed day) — biggest retention lever
- Daily challenge — song-of-the-day or "hit 40/min on F->G today"
- Earnable unlockables — themes/dial skins/chord-diagram styles (80s skins: neon, chrome, vaporwave)
- Growing plant/tree — a leaf per practice day

## Variable reward (use gently)
- Occasional random "golden chord" worth bonus sparkles
